🚀 Paso 0 — Antes de construir en Excel

Antes de crear en Excel, necesitas entender qué es Claude, cómo funciona dentro de Excel y cómo instalarlo correctamente. Esta guía te lleva desde cero hasta estar listo para crear proyectos profesionales con IA.

🤖 ¿Qué es Claude?

Claude es un modelo de inteligencia artificial creado por Anthropic, una empresa fundada por ex-investigadores de OpenAI. A diferencia de otros asistentes de IA, Claude destaca por:

  • Razonamiento profundo: puede analizar modelos financieros complejos, trazar dependencias entre celdas y explicar cálculos paso a paso.
  • Seguridad y transparencia: resalta cada cambio que hace, explica sus acciones y te pide confirmación antes de sobreescribir datos.
  • Contexto extendido: puede leer libros de trabajo completos con múltiples pestañas y entender las relaciones entre ellas.
Modelos disponibles en Excel: Puedes elegir entre Claude Opus 4.6 (más potente, ideal para tareas complejas) y Claude Sonnet 4.6 (más rápido, ideal para tareas cotidianas).

⚙️ ¿Cómo funciona Claude en Excel?

Claude funciona como un add-in (complemento) que se instala en Microsoft Excel. Aparece como una barra lateral dentro de la aplicación donde puedes chatear con Claude mientras trabajas en tu hoja de cálculo.

  • Lee tu workbook: Claude accede al contenido del libro de trabajo abierto — celdas, fórmulas, pestañas y estructura.
  • Ejecuta acciones nativas: puede modificar celdas, aplicar formato condicional, crear tablas dinámicas, gráficos y más.
  • Citaciones a nivel de celda: cada explicación incluye referencias clicables a las celdas específicas.
  • Protección: advierte antes de sobreescribir datos y puedes deshacer con Ctrl+Z como siempre.
💡 Atajo rápido: Presiona Ctrl+Alt+C (Windows) o Ctrl+Option+C (Mac) para abrir Claude desde cualquier lugar en Excel.

🚫 ¿Qué NO hace Claude en Excel nativamente?

Es importante conocer las limitaciones actuales para no frustrarte. Claude en Excel no puede:

  • Crear ni ejecutar Macros — no tiene acceso al editor de macros de Excel.
  • Escribir ni ejecutar código VBA — puede generar el código, pero debes copiarlo y pegarlo manualmente en el editor VBA.
  • Crear Data Tables (tablas de datos de análisis "What-If").
  • Guardar historial de chat entre sesiones — cada vez que abres el add-in, inicias una conversación nueva.
  • Funcionar en Excel 2016/2019 (licencias perpetuas) ni en Excel para Android.
  • Integrarse con auditoría o Compliance API empresarial (por ahora).
  • Acceder a archivos fuera del workbook abierto — solo lee lo que tienes abierto.
  • Reemplazar el juicio profesional — siempre requiere revisión humana, especialmente en entregables para clientes.
⚠️ Importante: Claude puede equivocarse. Nunca uses sus resultados como entrega final sin revisión. Tampoco uses Claude con archivos de fuentes desconocidas (riesgo de prompt injection).

📦 Cómo instalar el complemento en Excel

  1. Abre Claude for Excel en Microsoft Marketplace.
  2. Haz clic en "Get it now" (Obtener ahora).
  3. Abre Excel → pestaña Inicio → Add-ins (Complementos) y activa "Claude by Anthropic".
  4. Inicia sesión con tu cuenta de Claude en la barra lateral.
  5. ¡Listo! Ya puedes chatear con Claude dentro de Excel.
Versiones compatibles: Excel Web, Excel Windows (Microsoft 365), Excel Mac (16.46+), Excel iPad (2.51+).
No compatible: Excel 2016/2019 perpetuo, Excel en Android.

🎯 Primeros pasos: qué probar primero

Una vez instalado, prueba estos prompts para familiarizarte:

  • Explorar: "¿Qué hay en este libro de trabajo? Dame un resumen de cada pestaña."
  • Analizar: "¿Qué fórmulas hay en la columna D? Explícalas en español sencillo."
  • Limpiar: "Estandariza los formatos de fecha a AAAA-MM-DD en toda la hoja."
  • Crear: "Crea una tabla dinámica con ventas por mes y categoría."
  • Formatear: "Aplica formato condicional: verde si el valor es positivo, rojo si es negativo."
💡 Tip: Escribe / en la barra lateral para ver los Skills disponibles (flujos de trabajo predefinidos).

✅ Buenas prácticas para estudiantes

  • Siempre revisa los cambios antes de guardar — Claude puede equivocarse.
  • Sé específico en tus prompts: en lugar de "arregla esto", di exactamente qué quieres y en qué rango de celdas.
  • Usa instrucciones persistentes para definir tu estilo de trabajo (ej: "siempre usa formato IB").
  • Activa el Session Logging para que Claude registre todas sus acciones en una pestaña "Claude Log".
  • No uses archivos de fuentes desconocidas — podrían contener instrucciones ocultas maliciosas.
  • Administra tu uso: el add-in consume los límites de tu plan más rápido que el chat normal.
